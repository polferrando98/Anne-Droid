Anne Droid is a platformer game where the player has to reach two doors to survive. For the second assignment enemies use pathfinding(A*) to get to the player. For the third assignment there is UI implemented with score, timer, lifes and gears.

How to Play:
  • CONTROLS (KEYBOARD):
  • Move: A/D
  • Jump: W
  • 2Jump:WW
  • F1: Start from the first level
  • F2: Start from the current level
  • F5: Save
  • F6: Load
  • F8: Debug UI
  • F9: Debug Colliders
  • F10: God Mode
  • F11: Enable/Disable FPSCAP to 30

Cheats:

  • Cheats(KEYBOARD):
  • 1: LEVEL 1
  • 2: LEVEL 2
  • 3: LEVEL 3
  • 0: MENU

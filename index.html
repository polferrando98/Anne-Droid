Anne Droid

by Nina Lopez and Pol Ferrando.

The Game:

This is a project developed on the 2nd year of college at UPC.

Anne Droid is a platformer game where the player has to reach two doors to survive. Enemies use pathfinding(A*) to get to the player.

MEET THE TEAM:

How to Play:

  • CONTROLS (KEYBOARD):
  • Move: A/D
  • Jump: W
  • 2Jump:WW
  • F1: Start from the first level
  • F2: Start from the current level
  • F5: Save
  • F6: Load
  • F8: Debug UI
  • F9: Debug Colliders
  • F10: God Mode
  • F11: Enable/Disable FPSCAP to 30
